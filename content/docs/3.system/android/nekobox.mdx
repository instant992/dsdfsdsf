1. 1. Нажмите Скачать с GitHub в [боте](https://t.me/vpnghostbot), скачайте оттуда приложение и затем установите его.

![](https://vpnghost.space/hiddify/1.jpg)


2. Нажмите Добавить подписку в боте


3. Нажмите на три вертикальных линии ![](https://vpnghost.space/android/neko1.jpg) и выберите Конфигурации. ![](https://vpnghost.space/android/neko2.png)


4. Теперь осталось выбрать протокол и подключиться, нажав на иконку приложения внизу по центру. Готово!
![](https://vpnghost.space/android/neko3.png)
