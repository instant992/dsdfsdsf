1. Нажимем кнопку добавить подписку в [боте](https://t.me/vpnghostbot)

![](https://vpnghost.space/v2raytun/v2ray1.jpg)


2. Заходим в настройки по значку сверху справо и выбираем пункт VPN
![](https://vpnghost.space/v2raytun/v2ray2.jpg)


4. Готово, все настроено!


3. При желание можно отключить впн на ру сайтах или других. 
Для этого надо зайти в пункт "Сервисы" и выбрать там нужное, затем переподключиться к впн. 
![](https://vpnghost.space/v2raytun/v2ray3.jpg)
