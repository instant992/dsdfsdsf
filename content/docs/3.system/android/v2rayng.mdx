title: v2RayNG
description: Инструкции по настройке
---

1. Скачайте приложение через кнопку "Скачать с GitHub" в [боте](https://t.me/vpnghostbot), и установите его.


2. Нажмите кноку "Добавить подписку".

3. Нажмите на три вертикальных точки  ![](https://vpnghost.space/android/dots_android.jpg) и выберите "Обновить подписку" ![](https://vpnghost.space/android/update.jpg)

4. Теперь осталось выбрать протокол и подключиться, нажав на иконку приложения в правом нижнем углу. Готово!
![](https://vpnghost.space/android/sub.jpg)
